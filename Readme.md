# 🌱 AgroSustentável PR – Calculadora de Sustentabilidade Agrícola

## Concurso Agrinho 2026 | Programação | Subcategoria 3 – Front-End Web

---

## 🎯 Objetivo do Projeto

Este site foi desenvolvido para o **Concurso Agrinho 2026**, promovido pelo SENAR-PR e pela SEED-PR, com o tema:

> **"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

A proposta é uma **Calculadora de Sustentabilidade Agrícola** interativa, que permite ao produtor rural ou estudante informar dados sobre sua propriedade e receber um **índice de sustentabilidade personalizado** (de 0 a 100), com mensagens e dicas para melhorar suas práticas.

O projeto busca conscientizar sobre a importância do equilíbrio entre produção agrícola e preservação ambiental, especialmente no contexto do agronegócio paranaense.

## ⚙️ Funcionalidades

- **Calculadora interativa** com 7 variáveis de sustentabilidade
- **Índice de 0 a 100** calculado por critérios ponderados:
  - Área de preservação/mata nativa (até 30 pts)
  - Uso e gestão de água (até 25 pts)
  - Práticas sustentáveis: plantio direto, rotação, adubação, energia (até 45 pts)
- **Classificação em 4 níveis**: Início / Potencial / Em desenvolvimento / Sustentável
- **Mensagem personalizada** de acordo com o resultado
- **Modo claro/escuro** alternável pelo usuário
- **Animações de entrada** nas seções ao rolar a página
- **Totalmente responsivo** para celulares, tablets e computadores
- **6 dicas** práticas de sustentabilidade no agronegócio
